SASS

SASS es un preprocesador CSS diseñado para hacer CSS más fácil de mantener, hacer plantillas y extenderlo.

Sass se desarrolla en 2007 y es el preprocesador más utilizado.

SASS sobresale al utilizarse con frameworks como Bootstrap (la v4 sólo utiliza Sass) ofreciendo más funcionalidad

SASS permite el uso de dos sintaxis diferentes para crear tus archivos

  • SCSS (se parece más a un fichero CSS) → nueva
  • Sintaxis Sass (no utiliza llaves, ni puntos, ni comas y se basa en tabulaciones).
