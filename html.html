HTML

HTML es un lenguaje de marcas que permite describir páginas web. La página se dividirá en bloques que vienen definidos por etiquetas y es el propio explorador el que es capaz de interpretar las etiquetas para mostrarlas y darle el formato necesario.

HTML5 es la última versión de HTML (último estándar de 2014). El término representa dos conceptos diferentes:

  • Se trata de una nueva versión de HTML, con nuevos elementos, atributos y comportamientos.
  • Contiene un conjunto más amplio de tecnologías que permite a los sitios Web y a las aplicaciones ser más diversas y de gran alcance. A este conjunto se le llama HTML5 y amigos, a menudo reducido a HTML5.
  • HTML5 especifica dos variantes de sintaxis para HTML:
    • una «clásica», HTML (text/html), conocida como HTML5,
    • y una variante XHTML conocida como sintaxis XHTML5 que deberá servirse con sintaxis XML (application/xhtml+xml).

Un ejemplo de código en HTML5:

          <!DOCTYPE html>
            <html lang="es">
            <head>
              <meta charset="utf-8" />
              <meta name="viewport" content="width=device-width, initial-scale=1.0"/>
              <title>Ejemplo de HTML5
            </head>

            <body>
              <h1>Esto es un título
              <p>Esto es un párrafo

            </body>
            </html>
