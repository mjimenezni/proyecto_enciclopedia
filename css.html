CSS

CSS es un estándar de hojas de estilo en cascada que nos permite definir cómo se muestran los elementos en una página, informe, etc. Podemos crear distintos estilos para la misma página y un aplicar un mismo estilo a varias páginas.

CSS3 es la última evolución del lenguaje de las Hojas de Estilo en Cascada

CSS3 está en desarrollo desde 1999 , dividido en módulos más pequeños amplia la versión CSS2.

  • Esquinas redondeadas
  • Sonbras
  • Gradientes
  • Transiciones o animaciones
  • Nuevos layouts multi-columnas, cajas flexibles o grid layouts
