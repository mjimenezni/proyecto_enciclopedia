JAVASCRIPT

JavaScript es un lenguaje de programación elegido por la W3C como estándar para HTML5.

Es el único lenguaje que entienden los navegadores

JavaScript es un lenguaje de programación interpretado utilizado fundamentalmente para dotar de comportamiento dinámico a las páginas web

Un poco de historia:

  • Brendan Eich(1961-) : Desarrolla LiveScript en solo 10 días en 1995 para Netscape con el objetivo de dinamizar la web.
  • Estandarización (1996): ECMAScript o ECMA-26
  • Influyó en: JScript, ActionScript, .NET
  • W3C elige JavaScript como estándar para HTML5.
